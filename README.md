Project title: Bank Operations Processing

Описание проекта

Цель проекта: создать модуль обработки банковских операций клиента. Реализованы функции фильтрации по состоянию транзакций и сортировки по дате. Проект оформлен с использованием принципов GitFlow и PEP 8, с атомарными коммитами и примером README для удобной навигации.
Контекст: вы ведете разработку в репозитории на GitHub и используете ветки develop и feature/..., создавая пул-реквесты для код-ревью.
Установка

Требования:
Python 3.8+ (рекомендуется 3.10+)
Git
Установка зависимостей (при необходимости):
Создайте виртуальное окружение: python -m venv venv
Активируйте окружение: source venv/bin/activate ( Linux/macOS ) или venv\Scripts\activate ( Windows )
Установить зависимости можно через requirements.txt, если он есть:
pip install -r requirements.txt
Клонирование репозитория:
git clone <URL-репозитория>
cd <папка-репозитория>
